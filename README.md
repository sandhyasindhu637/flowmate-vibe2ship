## 📌 Problem Statement
PS1 — The Last-Minute Life Saver

Students frequently miss deadlines because existing tools give passive reminders that are easy to ignore. FlowMate goes beyond reminders — it takes meaningful action by autonomously planning, prioritizing, and helping students complete tasks before deadlines are missed.

## 🤖 What Makes FlowMate Different
Unlike every other to-do app, FlowMate doesn't wait for you to ask. Its autonomous AI agent runs on every page load, analyzes all your tasks, assigns risk scores, breaks high-risk tasks into subtasks, and tells you exactly what to do right now.

## ✨ Key Features
- **Autonomous AI Agent** — runs proactively without user prompting
- **Risk Scoring (0-100)** — each task gets an AI-calculated risk score
- **AI Brain Feed** — transparent log of every action the AI takes
- **Study Planner** — Gemini generates hour-by-hour timetable
- **Voice Input** — add tasks by speaking using Web Speech API
- **Google Calendar Integration** — export tasks as calendar events
- **Focus Mode** — Pomodoro timer with AI encouragement
- **Weekly Insights** — personalized productivity report by Gemini
- **"Will I finish on time?"** — honest AI prediction per task
- **Dark/Light Mode** — theme toggle with localStorage persistence
- **Student Profile** — context-aware AI suggestions per branch/semester
